northwind-php
=============

Microsoft Northwind Traders database SPA in HTML5/CSS/JavaScript/JSON/PHP/SQL.<br>
Using Kendo UI and jQuery front end to LAMP back end.

## Requirements

[kendo ui web](http://www.kendoui.com/download/download-kendo.aspx "Kendo UI WEB")
LAMP
[northwind](http://northwindextended.googlecode.com/files/Northwind.MySQL5.sql "Microsoft Northwind Traders database for MySQL"

This work is based on:

[example page 1](http://docs.kendoui.com/tutorials/PHP/build-apps-with-kendo-ui-and-php "Intro to Building Apps with Kendo UI And PHP Part 1") and [example page 2](http://http://docs.kendoui.com/tutorials/PHP/build-apps-with-kendo-ui-and-php-2 "2")
